Welcome To The Weather App

This weather app is a web-based application that allows the user to search for current weather information,weather
forecast of different cities around the world and a graph showing the change in the above parameters in the last 7 days of different cities around the
world.Users an search for weather information by city name.It uses the openweathermapAPI which is found at www.openweathermap.org
